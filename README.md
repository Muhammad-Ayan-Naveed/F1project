# F1 Race Analytics Dashboard

A data analysis tool for Formula 1 race data built in Python. Currently in progress.

## What It Does

Processes F1 race data including lap times, pit stop records, and driver performance 
across seasons using NumPy for numerical computation and SciPy for statistical analysis.

## Tech Stack
- Python
- NumPy
- SciPy

## Status
First version complete. Actively expanding features — additional seasons, 
visualizations, and performance comparisons in progress.

## How to Run
1. Install dependencies: `pip install numpy scipy`
2. Run: `python main.py` or just click the run button in vs code or the IDE you are using.
